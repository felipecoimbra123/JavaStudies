# Tipos de dados
Podem ser o nome de um aluno, quantidade de funcionários de uma empresa... Literalmente qualquer tipo de dado.

### Exemplos
* `Int` - Números inteiros. Ex: 12, -7, 12976
* `Double` - Números com vírgula. Ex: 1200.23; 3.07; 0.5
* `String` - Textos. Ex: "Sou o Felipe"
* `Char` - Textos de apenas 1 caractere (aspas simples). Ex: 'A', '@', '7'
* `Boolean` - Falso e verdadeiro. Ex: True ou False

# Variáveis
São os nomes simbólicos de endereços de memória, onde os dados serão armazenados.

### Exemplos
```java
double salario
```

Explicação: Especifica que 'salario' é o identificador, ou seja, o nome de uma variável que pode armazenar um valor real. 

```java
int n1
```

Explicação: Especifíca que 'n1' é o identificador de uma variável que pode armazenar um número inteiro

## Variáveis Constantes
São os valores que não mudam durante a execução do programa e nem de uma execução para outra.

### Exemplos
* `false`
* `Felipe Coimbra`
* `8.7`

# Expressões
As expressões podem utilizar operadores aritméticos, relacionais e lógicos. 

## Expressões Aritméticas
* `+` - Soma
* `-` - Subtração
* `*` - Multiplicação
* `/` - Divisão
* `%` - Resto da divisão

## Expressões Relacionais
* `>` - Maior
* `<` - Menor
* `==` - Igual
* `>=` - Maior ou igual
* `<=` - Menor ou igual
* `!=` - Diferente

## Expressões Aritméticas
* `&&` - E
